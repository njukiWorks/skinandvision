## Conversion Tracking Setup (Required Before Launch)

### Step 1: Create Google Ads Conversion Actions
In Google Ads → Tools → Conversions, create:
1. **Afspraak Aanvragen** (form submit on booking page) — value: €50
2. **Contactformulier Verstuurd** (contact form submit) — value: €30
3. **Telefoongesprek** (click on phone number) — value: €20
4. **Pagina: Tarieven Bekeken** (micro-conversion) — value: €5

### Step 2: Install Google Tag via GTM
- Tag type: Google Ads Conversion Tracking
- Trigger: Form submission confirmation / Thank-you page

### Step 3: Import GA4 Goals (optional)
- Link GA4 to Google Ads account
- Import Goal: Booking page visit
