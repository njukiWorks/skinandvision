## Pixel & Tracking Setup (Required Before Launch)

### Step 1: Create Meta Pixel
1. Meta Business Manager → Events Manager → Create Pixel
2. Pixel name: "Skin & Vision Clinic Pixel"
3. Install via GTM (recommended) or direct `<head>` tag

### Step 2: Standard Events to Fire
| Event | Trigger | Value |
|---|---|---|
| `PageView` | All pages | — |
| `ViewContent` | Treatment pages | — |
| `Lead` | Contact form submit | €30 |
| `Schedule` | Booking URL click | €50 |
| `Contact` | Phone click | €20 |

### Step 3: Conversions API (CAPI)
For iOS14+ attribution accuracy:
- Implement server-side event sending from Convex backend
- Use `fbq('track', 'Lead')` on client + Conversions API on server
- Deduplicate with `event_id` parameter

### Step 4: Test Pixel
- Meta Pixel Helper (Chrome extension) — verify events fire
- Events Manager → Test Events tab

## Creative Production Notes

### Images to Use from Existing Assets:
- Professional clinic photos → Authority/doctor ads
- Before/after eyelid photos → Result-focused ads (ensure patient consent)
- Clean clinic interior → Reels background B-roll

### Image Specs:
| Placement | Recommended Size | Aspect Ratio |
|---|---|---|
| Facebook Feed | 1080×1350 px | 4:5 |
| Instagram Feed | 1080×1080 px | 1:1 |
| Stories / Reels | 1080×1920 px | 9:16 |
| Carousel | 1080×1080 px | 1:1 |

### Copy Principles for Medical Ads:
- Never make clinical claims you cannot substantiate
- Use "kan helpen bij" not "geneest" or "verwijdert gegarandeerd"
- Include "Resultaten kunnen variëren" where showing before/after
- Meta's Special Ad Category may apply: select "Health & Beauty" not "Credit/Housing/Employment"
- **Do NOT** use superlatives like "Beste botox van Nederland" — Meta flags these
- Doctor credentials (BIG-registered) are strong trust signals — use prominently

## A/B Testing Plan (Months 2–3)

| Test | Variable A | Variable B |
|---|---|---|
| Hook | Safety angle ("Niet alle botox is gelijk") | Result angle ("Kraaienpootjes weg in 15 min") |
| CTA | "Maak een afspraak" | "Vraag gratis advies" |
| Creative | Static before/after | Doctor authority photo |
| Audience | Interest-based | Advantage+ Shopping Audience |
| Format | Single image | Carousel |

Run each test for minimum 7 days with equal budget split. Declare winner at 95% statistical confidence or after 50+ leads.
